# **App Name**: DebtEase & HomeStock

## Core Features:

- User Authentication: Secure user authentication using Firebase to manage debt tracking and home stock lists separately.
- Debt Tracking: A module for tracking debts including creditor details, amount, due dates, and payment history.
- Market Checklist: Create and manage a shopping checklist for items to purchase from the market.
- Restock Manager: Manage items needing to be restocked at home.
- Notifications: Reminders for debts due and low-stock alerts for home items.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calming and trustworthy feel.
- Background color: Light, desaturated blue (#F0F8FF) for a clean and airy interface.
- Accent color: Gentle green (#90EE90) to highlight successful transactions or additions to the home stock.
- Font: 'PT Sans', a humanist sans-serif for clear readability in both debt tracking and stock lists.
- Simple, outline-style icons for categories and actions to provide a user-friendly visual guide.
- Clean, card-based layouts for debts, shopping lists, and restock items to maximize usability and attractiveness, adaptable for mobile users.
- Subtle transitions and animations on updates to debt status or changes to the home stock lists.
